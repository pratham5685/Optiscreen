Installation Guide for Optiscreen

Hello User,

Thank you for showing interest in Optiscreen! I'm excited to help you enhance your browsing experience by reducing eye strain. Below is a step-by-step guide to install Optiscreen via a zip file:

Step-by-Step Guide to Install Optiscreen:

  1. Download the Zip File:

    Download the Optiscreen zip file from the link provided.

    Download Optiscreen Zip File

  2. Extract the Zip File:

    Locate the downloaded zip file on your computer.

    Right-click on the zip file and select "Extract All" or use any unzip tool to extract the contents to a folder.

  3. Open Chrome Extensions Page:

    Open Google Chrome.

    Click on the three dots in the upper-right corner of the browser to open the menu.

  4. Enable Developer Mode:

    In the Extensions page, toggle the Developer mode switch on. This is usually located at the top right of the page.

  5. Load Unpacked Extension:

    Click on the Load unpacked button.

    In the file dialog, navigate to the folder where you extracted the Optiscreen zip file.

    Select the folder (please select the root folder) and click Select Folder or Open.

  6. Verify Installation:

    Optiscreen should now appear in your list of installed extensions.

    You can pin it to your toolbar for easy access by clicking on the pin icon next to Optiscreen in the Extensions list.

Congratulations! Optiscreen is now installed and ready to use. You can now adjust brightness and blue light levels directly from your browser.
